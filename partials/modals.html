Help Menu

Using the command line, you can interact with this website and take control of your experience.

The terminal has a set of acceptable commands that will let you navigate the pages of the website and manipulate visible content.

In addition to this, the terminal emulates key convenience features such as using the up and down keys for navigating command history and the use of tab for autocompleting acceptable arguments and options.

Command Listing

Try out all the commands to see what they do.

Commands:  add cd git help ls man mv rm

Use the man command to learn more about how to use a specific command.

Example: Bring up the man page for the add command.

1. man add

Navigating Pages

Using the ls command, you can see available pages you can navigate to.

Choose one of them as an argument to the cd command to change the browser location.

Example: See available pages for navigation and then go to the resume page.

1. ls
2. cd resume

Manipulating Content

Using the add and rm commands, you can shape what information is being displayed on the view.

Use the tab key to help choose pieces of content.

Use of the . argument with add or rm command will add or remove all components, respectively.

Example: Change pages to the sandbox, add some components to the view, then clear everything.

1. cd sandbox
2. add navbar
3. add project tylersouthmayd.com
5. rm .
