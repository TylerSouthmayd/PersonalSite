TylerSouthmayd.com

July 2015 - Present

At the end of the Summer of 2015, I was armed with my RaspberryPi home web server and could finally start building out my ideas for my personal website.

The Idea? Let the user define what they see on my website through interacting with a command line interface (CLI) that I create.

I wanted to have a personal website, but going through the visual design phase was uninteresting to me. Eventually I came up with the idea that I didn't need to do the layout myself if I gave the option to the user.

Focusing more on building out the interactions with my CLI, I was able to implement data driven auto completion and command verification, the logic to navigate to different pages, emulate linux CLI features such as command history and man pages, and most importantly, to add and remove items from the view.
